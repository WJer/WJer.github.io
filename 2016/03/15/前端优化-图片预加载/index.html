前端优化-图片预加载

图片预加载

图片预加载就是图片提前加载，可以说是在牺牲前端的性能，因为你需要提前加载你所需要的所有图片，性能自然降低不少，但是这种做法能获得很好的用户体验，例如：图片轮播，轮播的过程中并不会出现卡顿的情况，因为图片加载完成之后才开始轮播的。

图片预加载的思路：主要是利用onload事件实现的。

Demo：

function preImgLoad (imgslist,callback) {
    //imgslist 预加载图片集合
    //callback 加载完一张图片的回执函数

    var images = [],
        progress = 0,
          _handle = function () {
              progress ++;
              if (progress === imgslist.length) {
                  callback();
              }
          };

    for (var i = 0,length = imgslist.length;i<length;i++) {
        images[i] = new Image();
        images[i].onload = images.onerror = _handle;
        images[i].src = imgslist[i];
    }
}
